# Clinical Therapy Robotics
**Developer: Lamya F. H. Ali**

This repository contains the source code for my graduation project involving the **Pepper Robot**.

## Project Overview
The project focuses on clinical therapy applications, utilizing computer vision and robotics simulation to assist in therapeutic environments.

## File Structure
- **pepper_v6.py**: The latest stable version of the therapy logic.
- **robot_bridge.py**: Integration layer for robot communication.
- **pepper_clinical_infinity_v3/v4/v5**: Development iterations.

## Requirements
- Ubuntu 22.04
- Python 3.10+
- OpenCV, MediaPipe, and PyBullet
